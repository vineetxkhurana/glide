Why bold subtitles improve watch time

Standard captions give every word the same visual weight. For viewers with ADHD or reading fatigue, that turns subtitles into a flat wall of text. Glide highlights the parts that matter, giving the eye natural anchor points so people stay with your content instead of drifting away.

Focus mode vs Calm mode

  • Focus ‚Äî NLP-driven partial-word bolding with adjustable intensity. Best for tutorials, lectures, and dense dialogue.
  • Calm ‚Äî bolds up to two full words per line for gentle, predictable reading. Best for vlogs, podcasts, and relaxed viewing.

How to make SRT subtitles bold automatically

Export captions from YouTube Studio or your editor as SRT, VTT, or ASS. Upload the file here, choose a mode, download the result, and re-upload to YouTube. The whole process takes under 30 seconds.

Built for subtitle accessibility

Over 360 million adults live with ADHD, and many more struggle with reading fatigue. Better subtitles aren't just about watch time ‚Äî they help real people follow along more comfortably.

Glide preserves all timing, tags, and formatting. Only the text changes. Your files stay fully compatible with every player.

API Documentation

Integrate Glide's subtitle processing into your application

Endpoint

POST https://glide-api-worker.glide-bionic.workers.dev/process

Request Body

{
  "text": "1\n00:00:00,000 --> 00:00:02,000\nYour subtitle text here",
  "format": "srt",
  "mode": "focus",
  "intensity": 0.5,
  "emphasisMode": "unicode"
}

Parameters

text string Required Subtitle content
format string Required srt, vtt, ass, or plain
mode string Required focus or calm
intensity number Optional 0.1 to 1.0 (default: 0.5, only for Focus mode)
emphasisMode string Optional unicode or html (default: unicode)

Response

{
  "processedText": "1\n00:00:00,000 --> 00:00:02,000\nùêòùê®ur ùê¨ùêÆbtitle ùê≠ùêûxt ùê°ùêûre"
}

Examples

JavaScript / Node.js

const response = await fetch(
  'https://glide-api-worker.glide-bionic.workers.dev/process',
  {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      text: subtitleContent,
      format: 'srt',
      mode: 'focus',
      intensity: 0.5
    })
  }
);

const { processedText } = await response.json();

Python

import requests

response = requests.post(
    'https://glide-api-worker.glide-bionic.workers.dev/process',
    json={
        'text': subtitle_content,
        'format': 'srt',
        'mode': 'focus',
        'intensity': 0.5
    }
)

processed_text = response.json()['processedText']
