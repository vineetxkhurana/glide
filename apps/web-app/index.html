API Documentation

Integrate Glide's subtitle processing into your application

Endpoint

POST https://glide-api-worker.glide-bionic.workers.dev/process

Request Body

{
  "text": "1\n00:00:00,000 --> 00:00:02,000\nYour subtitle text here",
  "format": "srt",
  "mode": "focus",
  "intensity": 0.5,
  "emphasisMode": "unicode"
}

Parameters

text string Required Subtitle content
format string Required srt, vtt, ass, or plain
mode string Required focus or calm
intensity number Optional 0.1 to 1.0 (default: 0.5, only for Focus mode)
emphasisMode string Optional unicode or html (default: unicode)

Response

{
  "processedText": "1\n00:00:00,000 --> 00:00:02,000\nùêòùê®ur ùê¨ùêÆbtitle ùê≠ùêûxt ùê°ùêûre"
}

Examples

JavaScript / Node.js

const response = await fetch(
  'https://glide-api-worker.glide-bionic.workers.dev/process',
  {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      text: subtitleContent,
      format: 'srt',
      mode: 'focus',
      intensity: 0.5
    })
  }
);

const { processedText } = await response.json();

Python

import requests

response = requests.post(
    'https://glide-api-worker.glide-bionic.workers.dev/process',
    json={
        'text': subtitle_content,
        'format': 'srt',
        'mode': 'focus',
        'intensity': 0.5
    }
)

processed_text = response.json()['processedText']

cURL

  -H 'Content-Type: application/json' \
  -d '{
    "text": "Your subtitle content",
    "format": "srt",
    "mode": "focus",
    "intensity": 0.5
  }'
