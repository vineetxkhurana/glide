Privacy Policy

Last updated: February 23, 2026

What Glide Does

Glide is a subtitle readability tool. It processes subtitle text to add bold emphasis on key syllables, making captions easier to read. Glide is available as a web app and a Chrome extension.

Data We Process

When you use Glide, subtitle text is sent to our API for processing. This includes:

  • Subtitle text — the raw caption content from your uploaded file (web app) or the active YouTube video (Chrome extension).
  • Processing settings — your chosen mode (Focus/Calm) and intensity level.

Data We Do NOT Collect

  • We do not store your subtitle text. It is processed in memory and discarded immediately.
  • We do not collect personal information, browsing history, or YouTube watch history.
  • We do not use cookies or tracking scripts.
  • We do not sell or share any data with third parties.

Chrome Extension

The Glide Chrome extension only activates on YouTube (youtube.com). It:

  • Reads subtitle data from the active video's caption track.
  • Sends subtitle text to our API (glide-api-worker.glide-bionic.workers.dev) for processing.
  • Stores your preferences (enabled state, mode, intensity) locally using chrome.storage.
  • Does not access any other websites, tabs, or browser data.

License Keys

If you purchase a license, your license key is validated via LemonSqueezy's API. We cache validation results temporarily (5 minutes) to reduce API calls. We do not store your payment information.

Rate Limiting

We use IP-based rate limiting to prevent abuse. Your IP address is used only for rate counting and is not stored permanently.

Hosting

Glide's API runs on Cloudflare Workers. The web app is hosted on Cloudflare Pages. Both are subject to Cloudflare's Privacy Policy.

Contact
